# Region map editor
### Not a modding tool!

Small Python app designed to edit maps in the style of Paradox and other grand strategy games, where every region is encoded with a unique color

## How to use
* Select a png, or put it in project folder with standard name
* Create an empty JSON file
* App will automatically scan your PNG file for unique regions and will fill the json with all the regions
* Start editing!
* * Change names
* * Add/Edit custom properties
* * Change color (TODO)

### Standard file names:
* "mapMask.png"
* "regions.json"

## How to build
Install and run main.exe, or build with any python builder, I used PyInstaller
